<h2>Sejarah singkat github</h2>
<p>

GitHub didirikan pada 2008 oleh Tom Preston-Werner, Chris Wanstrath, PJ Hyett, dan Scott Chacon** sebagai platform hosting kode berbasis Git. Tujuannya memudahkan kolaborasi dan manajemen versi kode secara online.

Pada 2018, GitHub diakuisisi oleh Microsoft senilai US$7,5 miliar, namun tetap beroperasi secara independen. Kini GitHub menjadi platform utama pengembang di seluruh dunia untuk open-source maupun proyek privat.
</p>

<h2>Cara kerja github</h2>
<p>GitHub bekerja sebagai platform penyimpanan dan kolaborasi kode berbasis Git. Kode disimpan di repository, perubahan dicatat dengan commit, disinkronkan lewat push dan pull, serta kolaborasi dilakukan melalui branch, pull request, dan issue.
</p>

<h2>Hubungan Github dengan Git</h2>
<p>Git adalah sistem version control untuk mengelola perubahan kode, sedangkan GitHub adalah platform online untuk menyimpan repository Git dan memudahkan kolaborasi pengembang.
</p>
